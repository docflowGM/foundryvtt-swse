SWSE SVG Kit

Starter SVG chrome kit for a cold-steel / blue-holo datapad UI.

Files:
- frame-shell.svg: outer application frame
- panel-large.svg: large content panel
- panel-small.svg: small side panel
- header-bar.svg: top header/mentor bar chrome
- footer-bar.svg: footer/command strip chrome
- button-frame-primary.svg: primary action button frame
- button-frame-secondary.svg: secondary action button frame
- mentor-ring.svg: circular portrait ring/overlay
- tree-preview.svg: sample node-map/tree preview
- modal-frame.svg: modal shell/frame

Usage suggestion:
- Use SVG for geometry/chrome
- Use CSS for glow, pulse, hover, tint, scanlines
- Keep painted art and portraits as PNG/WebP
