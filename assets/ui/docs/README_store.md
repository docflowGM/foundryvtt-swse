# SWSE Store SVG Pack

Dedicated holo SVG assets for the store / exchange UI.

## Intended surfaces
- Store shell and splash layer
- Wallet / credits / cart sidebar
- Search, filter, category, sort bar
- Portraitless product cards and listing rows
- Glyph/Aurebesh-style item presentation
- Checkout / rarity / stock / discount widgets
- Future splash screen pieces matching chargen

## Notes
- These assets are intentionally de-texted
- Real labels, prices, names, and localization should be HTML
- `splash/` is for the future intro experience once chargen is settled
