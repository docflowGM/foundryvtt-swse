# SWSE Holo Wave 2 Modular SVG Kit

This pack focuses on composable overlays and inner SVG parts rather than full outer frames.

Included groups:
- headers: title/subtitle/badge strips
- meters: segmented, continuous, dot, threshold, spent overlays
- overlays: table/row/header effects
- cards: card hand/discard/state overlays
- portrait: frame/mask/scanline/glow/badge layers
- tree: talent node, connectors, active/locked states
- tabs: state variants
- widgets: stars, checks, chips, search, numeric badges

Intended use:
- place in `assets/ui/`
- combine with existing page/partial frames from the main holo kit
- layer via CSS background-image, pseudo-elements, or nested wrappers
