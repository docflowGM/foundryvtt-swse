# SWSE Holo SVG Wave 4 — Ship & Droid Focus

This pack adds interaction/state-heavy assets for starship and droid interfaces.

## Included
- Ship subsystem cards: 8
- Ship arc overlays: 8
- Ship crew markers: 12
- Ship status overlays: 8
- Droid body-slot frames: 8
- Droid socket-state overlays: 8
- Droid processor/logic plates: 8
- Droid power-state assets: 8
- Shared system state assets: 8

## Intended use
These SVGs are designed to sit on top of, or inside, larger holo panels from earlier waves.
They avoid baked-in text and are safe for Foundry HTML overlays.
