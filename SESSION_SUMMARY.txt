HOW TO USE (COPY & PASTE)
================================================================================

1. Load Foundry with system
2. Open browser console (F12 → Console)
3. Paste this command:

    const sheet = document.querySelector('.swse-sheet');
    const report = SentinelTabDiagnostics.diagnose(sheet);
    console.log('SEVERITY:', report.summary.severityLevel);
    console.log('ISSUES:', report.summary.issues);
    console.log('CSS RULES:', report.diagnostics.cssRules.suspiciousRules);

4. Check the severity level:
   - OK: Tabs working ✅
   - WARNING: Suspicious CSS but functional
   - ERROR: Tabs broken - CSS is culprit
   - CRITICAL: No tab panels found

5. If ERROR or CRITICAL, review "CSS RULES" output
   This shows EXACTLY what CSS is breaking tabs
