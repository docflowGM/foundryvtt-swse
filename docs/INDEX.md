# Documentation Index

## 📚 Documentation Structure

This folder contains comprehensive documentation for the SWSE Foundry VTT System. Files are organized by category below.

### 🚀 **Getting Started**
Start here if you're new to the system.
- **guides/** - User guides, quick start, and contribution guidelines
- **setup/** - Installation and configuration instructions
- **references/** - General information (README, FAQ, Features)

### 🏗️ **Architecture & Design**
For developers and architects understanding the system design.
- **architecture/** - System architecture, data models, and design documents
- **data-models/** - Data structure specifications and schemas

### ⚙️ **Systems & Features**
Documentation for specific game systems.
- **systems/** - Mentor, Combat, Progression, Talents, Force, Droids, Vehicles, etc.
- **governance/** - Rules governance and compliance
- **store/** - Equipment and commerce system documentation

### 📊 **Operations & Maintenance**
- **reports/** - Analysis and diagnostic reports
- **audit/** - Audit findings and compliance documentation
- **migrations/** - Data migration guides and scripts
- **tools/** - Tool documentation and utilities

### 📦 **Reference Material**
- **swse-classes/** - SWSE class system reference
- **templates/** - Template system documentation

### 📁 **Archives**
- **_archive/** - Old session reports and superseded documentation
