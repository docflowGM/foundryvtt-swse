
# Update Your Character Sheet Template

Your templates need `data-action` attributes to work with Foundry V2 sheets.

## Location
C:\Users\Owner\Documents\GitHub\foundryvtt-swse\templates\actors\character\character-sheet.hbs

## Find and Replace

### Change 1: Add Weapon Button
**Find:**
```handlebars
<button type="button" class="add-weapon btn-add">
```

**Replace with:**
```handlebars
<button type="button" class="add-weapon btn-add" data-action="addWeapon">
```

## Pattern
All buttons need: `data-action="actionName"`

The action name should match the method in the JavaScript class.
